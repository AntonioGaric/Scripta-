Dodavanje CSS-a u HTML

Što je CSS?
CSS služi za oblikovanje HTML elemenata (boje, fontovi, raspored...)

Načini dodavanja:

  • Inline – unutar taga pomoću style
  • Internal – u style dijelu
  • External – u posebnoj .css datoteci
Preporučuje se koristiti external CSS jer se isti stil može koristiti na više stranica.

Ključni elementi koje smo učili

  • – glavni kontejner za sve HTML elemente
  • – sadrži meta informacije o stranici
  • – sadrži vidljivi sadržaj stranice
  • h1 - h6 – naslovi različitih veličina
  • p – paragrafi teksta
Važno: HTML tagovi se uvijek zatvaraju! Svaki otvoreni tag mora imati svoj zatvoreni tag (osim self-closing tagova poput

CSS Opacity

Što je opacity?
Svojstvo opacity određuje prozirnost elementa. Vrijednosti idu od 0 (potpuno nevidljivo) do 1 (potpuno vidljivo).

Napomena: Kada koristiš opacity na elementu, svi njegovi unutarnji elementi (tekst, slike...) također postaju prozirni. Ako želiš da samo pozadina bude prozirna, koristi rgba() boje.
